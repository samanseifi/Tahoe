Current Capabilities
 

Console Root Commands
Command Arguments Description
  AddBody 2 arguments: ordered
        (unnamed) (string) (no default) : path to database file
        (unnamed) (string) (default = "auto") : database file format		 Input data files to be displayed.  The body is automatically added to the 0.0.frame.  Currently each file must be of type Exodus ouput(.exo, .e, .geo) and inputted individually.  The database files can also be inputted when the application is started up using the -f flag, e.g. vtk_console -f big.exo -f square0CSE.exo.
  FlipBook 2 arguments: not ordered
        delay (double) (default = 0) : frame delay in seconds
        step (integer) (default = 1) : time step increment		 Show flip book of images from each time step.  This occurs in all the frames in the window.
 HideFrameNumbers 0 arguments Hide frame numbers for each corresponding frame.
 Interactive 0 arguments Allow manipulation of the bodies in the display via the mouse. The left button controls rotation of the body, the middle button controls panning, and the right button controls zooming in and out. Other keyboard commands are available such as: 
      key press 'j'/key press 't' - toggle between joystick (position sensitive) and trackball (motion sensitive) styles 
      key press 'w'/key press 's' - toggle between wire frame and surfaces for bodies 
      key press 'p' - displays  marker on body along with node number, coordinates and scalar value of node that 
                                mouse is currently pointing to
      key press 'e' - exit interactive mode
 Layout 2 arguments: not ordered
        nx (integer) (no default) : number of horizontal frames
        ny (integer) (no default) : number of vertical frames		 Set up the number of frames lengthwise (nx) and widthwise(ny) of main window.
 NextTimeStep 1 argument
        (unnamed) (integer) (default = 1) : number steps forwards		 Move dispalyed data in all frames forward by inputted number of time steps.
 Pan 2 arguments: not ordered
        x (double) (default = 0) : x-direction pan (dx/w)
        y (double) (default = 0) : y-direction pan (dy/h)		 Enter command line argument for panning/translating bodies in x and y directions in all frames. 
 Point 0 arguments Display bodies as points in all frames as opposed to wireframes or surfaces.
 PreviousTimeStep 1 argument
        (unnamed) (integer) (default = 1) : number steps backwards		 Move displayed data in all frames backward by inputted number of time steps.
 RemoveBody  1 argument
        (unnamed) (integer) (no default) : body number to remove		 Remove one of data files that is currently being displayed.
 ResetView 0 arguments Reset camera to initial default view in all frames.
 Rotate 3 arguments: not ordered
        x (double) (default = 0) : x-axis rotation
        y (double) (default = 0) : y-axis rotation
        z (double) (default = 0) : z-axis rotation		 Enter command line argument (in degrees) for rotating bodies in x, y, and z directions in all frames.  This can be a bit tricky since it rotates on permanent x, y, z axes and not on the current axes. 
 Save 2 arguments: ordered
        (unnamed) (string) (no default) : image file name
        (unnamed) (string) (default = "JPG") : image file format (TIFF|JPG|PS)		 Save an image of the main window as a .tif, .ps,  or .jpg.
 SaveFlipBook 2 arguments: ordered
        (unnamed) (string) (no default) : image file name
        (unnamed) (string) (default = "JPG") : image file format (TIFF|JPG|PS)		 Save multiple images from a flip book as a series of .tif, .ps, or .jpg.  The flip book images can be combined to form .mov quicktime animations.
 SelectTimeStep 1 argument
        (unnamed) (integer) (no default) : time step		 Select time step from given range to be displayed in all frames.  When multiple bodies with different time ranges are in the same frame, selecting a time step becomes a bit tricky.  This issue shall be resolved in the future.
 ShowFrameNumbers 0 arguments Show frame numbers for each corresponding frame.
 Surface 0 arguments Display bodies as solid surfaces in all frames as opposed to wireframes or points.
 Update 0 arguments Update displays in all frames after changes have been made to data.
 WindowSize 2 arguments: not ordered
        w (integer) (no default) : width
        h (integer) (no default) : height		 Set size, in pixels, of main window.
 Wire 0 arguments Display bodies as wireframes in all frames as oppposed to surfaces or points.
 Zoom 1 argument
        (unnamed) (double) (no default) : zoom factor		 Zoom camera in/out in all frames by inputtted zoom factor.

Body Commands
 
Command Arguments Description
 HideAxes 0 arguments Hide x,y,z axes display in the frame.
 HideColorBar 0 arguments Hide scalar color bar.
 HideContours 0 arguments Hide contour surfaces/lines for the body.
 HideCuttingPlane 0 arguments Hide all cutting planes for the body.
 HideElementNumbers 0 arguments Hide element number displays for the body.
 HideGlyphs 0 arguments Hide vector glyphs for the body.
 HideNodeNumbers 0 arguments Hide node number dispalys for the body.
 Interactive 0 arguments Allow manipulation of the bodies in the display via the mouse. The left button controls rotation of the body, the middle button controls panning, and the right button controls zooming in and out. Other keyboard commands are available such as: 
      key press 'j'/key press 't' - toggle between joystick (position sensitive) and trackball (motion 
                                                      sensitive) styles.
      key press 'w'/key press 's' - toggle between wire frame and surfaces for bodies 
      key press 'p' - displays  marker on body along with node number, coordinates and scalar value of 
                                 node that mouse is currently pointing to.
      key press 'e' - exit interactive mode.
 Pick 1 argument
        node (integer) (default = 1) : node number		 Display marker on body along with coordinates and scalar value of specified node.
 Point 0 arguments Display body as points in the frame as opposed to a wireframe or surface.
 Rotate 3 arguments: not ordered
        x (double) (default = 0) : x-axis rotation
        y (double) (default = 0) : y-axis rotation
        z (double) (default = 0) : z-axis rotation		 Enter command line argument (in degrees) for rotating all bodies in x, y, and z directions in the frame.  This can be a bit tricky since it rotates on permanent x, y, z axes and not on the current axes.
 ShowAxes 0 arguments Show x,y,z axes display for the body.
 ShowColorBar 2 arguments: ordered
        (unnamed) (string) (default = "<DEFAULT>") : body to reference scalar range
        location (string) (default = "L") : color bar position, where T/t = top, L/l = left, B/b = bottom, R/r = right		 Show scalar color bar in specified location of the frame.
 ShowContours 0 arguments Display body with contours.  Contours are displayed as surfaces in the 3-D case, and lines in the 2-D case.
 ShowCuttingPlane 6 arguments: not ordered
        oX (double) = 0.5 (default = 0) : x-coordinate of origin
        oY (double) = 0.5 (default = 0) : y-coordinate of origin
        oZ (double) = 0.2 (default = 0) : z-coordinate of origin
        nX (double) = 0 (default = 0) : x-coordinate of normal
        nY (double) = 0 (default = 1) : y-coordinate of normal
        nZ (double) = 1 (default = 0) : z-coordinate of normal		 Display 3-D body with specified cutting plane.  Multiple cutting planes can be added.  This can be combined with the "ShowContours" command to show cutting planes with contour lines.
 ShowElementNumbers 0 arguments Show element number displays for the body.
 ShowGlyphs 4 arguments: not ordered
        filter (boolean) (default = true) : Show only visible points (true/false)
        base (string) (default = "tail") : Location of base of arrow (head/tail)
        scale (boolean) (default = true) : determines whether the vectors scale in size or are of fixed length (true/false)
        color (boolean) (default = true) : color glyphs by values or not (true/false)
Display vector glyphs on the body.
 ShowNodeNumbers 0 arguments Show node number displays for the body.
 Surface 0 arguments Display body as a solid surface as opposed to a wireframe or points.
Update 0 arguments Update body display after changes have been made to data.
 Wire 0 arguments Display body as wireframe as oppposed to a surface or points.


 
 

Body Variables
 
Variable Description
 min_Scalar_Range Minimum scalar range value
 max_Scalar_Range Maximum scalar range value
 numColors Number of colors used to display the body.  Default is 256 colors.
 scale_factor Scale factor for displacement vector.  Scale factor from 0 to 1 decreases displacements.  Scale factor greater than 1 increases displacements.  Default is 1.
 opacity Opacity of body.  Range is from 0 (transparent) to 1 (opaque).  Default is 1.
 numContours Number of contours to be used with the "ShowContours" command.  Default is 10.
