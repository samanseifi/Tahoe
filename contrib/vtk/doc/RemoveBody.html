RemoveBody

Console Root Command
 
Remove a data file from all the frames.

Ex:  vtk> Remove
        range 0 to 2
        body number to remove: 2
        body 2 removed from 1 frame
 


 


Frame Command
 
Remove a body from the frame.

Ex: 0.0.frame:  > Rem
      body to remove (0,1)
      body number: 1
